## Syntax Highlighting

### HTML

`HTML syntax highlighting`

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>Title Of web page</title>
  </head>
  <body>
    <!-- Content Of web page -->
  </body>
</html>
```

> If you hover on previous block of code, copy icon will appear, just click on it to copy the block of code content
